Fragen und Antworten
Können wir unsere Sponsoren im System anzeigen?
                              
                                Ja, integrieren Sie beliebig viele Sponsoren, diese können auf dem Login, dem Dashboard und der Platzbelegung angezeigt werden. Diese Möglichkeit ihre eigenen Sponsoren einzubinden können Sie als Refinanzierung unseres Online Buchungssystem nutzen.
                            
Lassen sich wiederkehrende Termine eintragen?
                              
                                Tägliche/Wöchentliche oder 14-tägige Trainingseinheiten können ganz einfach als Serientermin für einen bestimmten Zeitraum angelegt werden, aber auch Abonnements können angelegt und verwaltet werden.
                            
Kann ich auch ein einzelnes Modul noch zu unserem System hinzubuchen?
                              
                                Die Basisvariante unseres Online Platzbuchungssystem ist die Light-Version, ein Wechsel zu einer anderen Version ist natürlich jederzeit möglich. Einzelne Module aus einer Version können auf Wunsch auch 28 Tage lang getestet werden und daraufhin auch einzeln gebucht werden.
                            
Kann ich die Einstellungen des Systems selbst verwalten?
                              
                                Ja! Administratoren können alle Einstellungen selbst verwalten und auch jederzeit anpassen. Konfigurieren Sie sich Ihr System so, dass es perfekt zu Ihrem Verein passt. Bei Fragen helfen wir Ihnen immer sehr gerne weiter!
                            
Müssen alle Mitgliedsdaten manuell in das Buchungssystem eingepflegt werden?
                              
                                Nein. Mithilfe unseres Mitgliederimports ist es ganz einfach, diese selbst in das System zu importieren. Wenn gewünscht, übernehmen wir den Import gerne für Sie!
                            
Wie lange dauert die Einrichtung des Systems?
                              
                                Innerhalb von 48 Stunden nach Vertragsabschluss können Sie mit courtbooking.de starten! Sie bekommen Ihre eigene Domain "IhrVerein.courtbooking.de" und erhalten einen eigenen Email-Zugang "IhrVerein@courtbooking.de".
                            
Können sich auch Gäste im System anmelden und können diese Gaststunden auch abgerechnet werden?
                              
                                Selbstverständlich. Sie legen fest, wer sich in Ihrem System registrieren darf. Alle Gast- sowie kostenpflichtigen Stunden lassen sich problemlos per SEPA-Lastschrifteinzug abrechnen.
                            
Benötigt jedes Mitglied eine eigene E-Mail Adresse?
                              
                                Nein. Eine E-Mail Adresse kann von mehreren Mitgliedern verwendet werden. So können z.b. Familienmitglieder die gleiche Email-Adresse benutzen.
                            
Gibt es nur einen Admin im System?
                              
                                Nein. Es ist eine beliebige Anzahl an Administratoren möglich. Weiterhin können Sie verschiedene Mitgliedsarten erstellen und denen unterschiedliche Rechte zuweisen.
                            
Gibt es eine Anleitung für courtbooking.de?
                              
                                Ja, es gibt eine Onboarding Präsentation, die die wichtigsten Informationen für den Einstieg mit Ihrem neuen System enthält. Zusätzlich dazu gibt es eine Hilfe im System, in der alle Funktionen beschrieben sind.
