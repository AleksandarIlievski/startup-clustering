						Data transfer					
						The LiFi technology is to use wireless data which is embedded in its beam.					
 
						getting data					
						A LiFi enabled device converts the ray of light into an electric signal which is  converted back into data.					
 
						using data					
						The device sends back data using invisible light. LiFi provides high speed bidirectional light communication networks.
